# Verilog 电梯控制器

一个用 Verilog HDL 实现的多层电梯控制系统，支持内部请求和外部请求，采用状态机设计实现智能调度。

## 运行说明

### 编译和仿真

使用 Icarus Verilog 进行仿真：

```bash
# 编译设计文件和测试台
iverilog -o elevator_sim elevator_controller.v elevator_tb.v

# 运行仿真
vvp elevator_sim
```

## 测试结果

所有测试场景均成功通过：

### 测试 1：内部请求前往 2 层 ✓
- 电梯从 0 层上行到 2 层
- 到达后自动开门，保持一段时间后关门

### 测试 2：外部上行请求从 1 层 ✓
- 电梯从 2 层下行到 1 层
- 到达后自动开门，保持一段时间后关门

### 测试 3：外部下行请求从 3 层 ✓
- 电梯从 1 层上行到 3 层
- 到达后自动开门，保持一段时间后关门

### 测试 4：多个请求（1 层和 3 层）✓
- 电梯先下行到 1 层
- 然后上行到 3 层
- 每次到达都正确开门和关门

## 项目特性

- 支持 4 层电梯（可配置）
- 内外部请求处理
- 方向优先调度算法
- 自动门控系统
- 兼容标准 Verilog
